come up with a plan to handle reviews from bots.

bots have names that include "\[bot\]".

bot reviews should not affect the overall approval status. if a PR was
approved by a bot, it should have a bot icon followed by a check icon in
the review pill. if a bot commented, it should show a bot icon followed
by a speech bubble icon. if a bot requested changes, it should show a
bot icon followed by an x icon.

write your plan to `PLAN.md` in this directory.
